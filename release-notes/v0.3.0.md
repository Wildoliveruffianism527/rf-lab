## ✨ What's New

### NRF24 Jammer (new!)

Two transmit modes covering the full 2.4 GHz band:

- **CW** — constant wave at any user-selected channel (0–125, i.e. 2400–2525 MHz). ←/→ navigates with ±1 / ±5 step.
- **Sweep** — full-band channel hop across all 126 channels.

Maximum transmit power for effective interference testing.

### Main menu

The app now lands on a main menu after the splash screen, letting you choose between **Scanner** and **Jammer** mode. Pressing Back from either mode returns you to the menu instead of exiting the app.

### Continued from v0.2.0

- 126-channel real-time NRF24 spectrum scanner
- WiFi 1/6/11 + BLE 37/38/39 frequency markers
- Adjustable dwell time (130–2000 µs)
- Cursor inspection with live frequency / hit-count readout
- Auto max-hold mode

## 📦 Install

### Easiest — qFlipper drag-and-drop

1. Download `pingequa_rf_toolkit.fap` from the **Assets** section below
2. Connect Flipper to qFlipper, drag the file into `/ext/apps/GPIO/`
3. On Flipper: `Apps → GPIO → PINGEQUA RF Lab`

### Power user — ufbt

```bash
git clone https://github.com/pingequa/pingequa_rf_toolkit
cd pingequa_rf_toolkit
git checkout v0.3.0
ufbt launch
```

## 🎮 Quick Usage

After splash, choose Scanner or Jammer:

**Scanner**:
- ← / → cursor (hold = ±5)
- ↑ / ↓ dwell
- OK pause/resume; Back → menu
- Auto MAX HOLD on saturation

**Jammer**:
- Choose CW or Sweep
- ← / → channel (CW); OK toggle TX; Back → menu

Detailed walkthrough: [docs/QUICKSTART.md](https://github.com/pingequa/pingequa_rf_toolkit/blob/v0.3.0/docs/QUICKSTART.md).
