# PINGEQUA RF Lab — Press Kit

## One-Line Pitch (10 words)

Open-source 2.4 GHz spectrum analyzer for Flipper Zero, plug-and-play.

## Short Pitch (60 words)

PINGEQUA RF Lab is an open-source 2.4 GHz spectrum analyzer + NRF24 jammer for Flipper Zero, designed for the [PINGEQUA 2-in-1 RF Devboard](https://www.pingequa.com/products/flipper-zero-nrf24-cc1101-2-in-1-rf-devboard). It works plug-and-play across every major Flipper firmware (Momentum, Unleashed, RogueMaster, Official) where legacy single-chip NRF24 scanner apps fall short. MIT licensed.

## Medium Pitch (200 words)

Flipper Zero is a popular pen-test and IoT hacking device, but its onboard radio only covers Sub-GHz. For 2.4 GHz work — WiFi audit, BLE discovery, RC drone analysis, NRF24 research — owners need an external module.

The PINGEQUA 2-in-1 RF Devboard is a dual-chip expansion module with both nRF24L01+ and CC1101 in one package. One board, both bands, software-switched.

But this dual-chip design breaks legacy NRF24 scanner apps written for single-chip boards — failed scans, broken Sub-GHz access after exit, firmware-specific tweaks needed.

**PINGEQUA RF Lab** is built clean-sheet for the dual-chip era. The result is a real-time spectrum analyzer plus NRF24 jammer that runs plug-and-play on Momentum, Unleashed, RogueMaster, and Official Flipper firmware — no firmware patching, no broken apps.

Features include max-hold mode, WiFi 1/6/11 + BLE 37/38/39 band markers, microsecond-resolution dwell tuning, cursor channel inspection, and CW + Sweep jamming modes (v0.3.0+).

Open source MIT. 28 KB FAP. Continuous releases.

## Key Differentiators

- ✅ **Specifically designed for the dual-chip 2-in-1 board** — others assume single chip
