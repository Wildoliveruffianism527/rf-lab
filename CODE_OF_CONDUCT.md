# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) — short version:

- Be respectful and constructive
- Welcome newcomers, regardless of experience level
- Focus criticism on code and ideas, not people
- Assume good intent; ask for clarification before assuming malice
- Don't share private communications without consent
- Don't harass, threaten, or discriminate

## Reporting

Conduct violations: [support@pingequa.com](mailto:support@pingequa.com).

Reports are reviewed privately. Reporters' identities are kept confidential when possible. We may take any action up to and including banning offenders from the project's GitHub organization.

## Scope

This Code of Conduct applies to all project spaces — issues, discussions, pull requests, code comments, project email — and to anyone representing the project in public.
